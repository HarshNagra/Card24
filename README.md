# Card24
 Card24 game using Android Studio Basics
 
 ### Java - Android Studio
 
 ### Libraries Used
 JPE to perform the calculations. 
 
 ### How to play - 
 1. Run the app.
 2. Enter a number.
 3. Manipulate the numbers on the 4 poker cards using the given arithmetic operations such that they equal the number you inputted.  
 4. Get the result.
